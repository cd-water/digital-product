# 数码产品在线购物系统智能客服知识库

## 一、用户登录与注册相关
### 1.1 账号登录
- **问题**：如何通过账号登录？
- **答案**：
  1. 打开数码产品在线购物电商系统首页，点击登录入口进入登录页面；
  2. 选择“账号登录”方式；
  3. 填写信息：
     - 账号：5-20位字母、数字或下划线组合；
     - 密码：8-32位，含大写字母、小写字母、数字和特殊字符（如！@#等）；
     - 选择身份角色（普通用户/店铺主/管理员）；
  4. 可选勾选“记住我”实现快速登录；
  5. 点击登录，验证通过后跳转至对应角色页面；
  6. 未注册可点击“注册”链接，或选择“游客访问”。

### 1.2 手机号登录
- **问题**：如何通过手机号登录？
- **答案**：
  1. 打开登录页面，切换至“手机号登录”（非默认方式）；
  2. 输入注册绑定的11位有效手机号（如13、15开头），选择登录角色；
  3. 点击“获取验证码”，手机接收6位数字短信（手机号错误则收不到）；
  4. 验证码60秒内不可重复获取，输入后点击登录；
  5. 验证通过后跳转至对应角色页面；
  6. 收不到验证码时，检查手机号正确性及短信拦截设置。

### 1.3 账号注册
- **问题**：如何注册账号？
- **答案**：
  1. 首页点击注册入口进入注册页面；
  2. 填写信息：
     - 账号：5-20位字母、数字或下划线（无其他字符）；
     - 密码：8-32位，含大写字母、小写字母、数字、特殊字符中至少三种；
     - 确认密码：与密码一致；
     - 身份角色：选择“普通用户”或“店铺主”（暂不支持管理员直接注册）；
  3. 点击“注册”，系统验证通过后跳转至登录页；
  4. 可直接用新账号登录，或点击“登录”返回登录页。

### 1.4 密码重置
- **问题**：忘记密码后如何重置密码？
- **答案**：
  1. 登录页面点击“忘记密码”进入找回页面；
  2. 填写注册绑定的11位手机号，选择对应角色；
  3. 点击“获取验证码”（60秒冷却），输入收到的6位短信验证码；
  4. 设置新密码：
     - 8-32位，含大写字母、小写字母、数字、特殊字符中至少三种；
     - 确认新密码需与设置一致；
  5. 点击“确认”，验证通过后跳转至登录页，可用新密码登录。

### 1.5 手机号注册限制
- **问题**：一个手机号能否注册多个账号？
- **答案**：
  1. 系统采用“手机号唯一绑定”规则，一个手机号仅能注册一个账号；
  2. 多账号需使用不同手机号分别注册；
  3. 每个手机号注册时需独立接收验证码，确保账号归属清晰。


## 二、商品浏览相关
### 2.1 轮播图跳转
- **问题**：首页的轮播图点击后会进入哪里？
- **答案**：首页轮播图展示热门数码产品，点击后跳转至对应数码产品详情页，可查看详细信息、价格、库存等。

### 2.2 特定数码产品类型列表入口
- **问题**：如何从首页快速进入特定类型的数码产品列表？
- **答案**：
  1. 首页左侧“热门数码产品类型”区域有“手机”“笔记本”等标签；
  2. 点击目标标签，直接跳转至该类型数码产品的列表页。

### 2.3 “大家都在搜”功能
- **问题**：首页的“大家都在搜”板块有什么作用？
- **答案**：展示用户常搜索的数码产品名称，点击名称链接可跳转至对应数码产品详情页，快速查看热门数码产品信息。

### 2.4 系统公告查看
- **问题**：如何查看系统公告的完整内容？
- **答案**：
  1. 首页右侧“系统公告”区域显示最新公告标题；
  2. 点击标题右侧“更多”按钮，弹窗展示公告完整标题和内容。

### 2.5 首页服务信息
- **问题**：首页能直接获取哪些服务信息？
- **答案**：
  - 客服信息：在线客服7×24小时服务，电话400-XXX-XXXX；
  - 物流信息：全国包邮（部分偏远地区除外），支持多种配送方式；
  - 售后保障：7天无理由退换货，正品保障。

### 2.6 全量列表入口
- **问题**：从首页如何进入更多数码产品或数码配件的列表页？
- **答案**：
  1. 首页“推荐数码产品”板块右侧点击“更多数码产品”，进入全部数码产品列表页；
  2. 首页“热销数码配件”板块右侧点击“更多数码配件”，进入全部数码配件列表页。

### 2.7 “热门店铺”信息
- **问题**：首页的“热门店铺”板块能直接查看哪些信息？
- **答案**：
  - 店铺头像和名称；
  - 详细地址（省市区+具体地址）；
  - 联系电话；
  点击店铺信息可跳转至详情页。

### 2.8 列表页功能（筛选/搜索/加载）
- **问题**：数码产品/数码配件列表页支持哪些筛选、搜索和加载功能？
- **答案**：
  1. 筛选功能：
     - 数码产品列表：按“数码产品类型”“仅看有货”“价格区间（0-10000元，含快捷档位）”筛选；
     - 数码配件列表：按“数码配件类型”“仅看有货”“价格区间”筛选；
     - 店铺列表：按“省市区地区”“店铺昵称”筛选；
  2. 搜索功能：
     - 支持输入名称（数码产品/数码配件/店铺名）回车或点击“搜索”精准查找；
     - 店铺支持地区级联筛选；
  3. 加载功能：
     - 数码产品/数码配件列表：无限滚动加载（滚动至底部100px自动加载下一页）；
     - 店铺列表：分页控件（可调整页码、每页条数，支持12/24/48/96条切换）；
  4. 重置功能：所有列表页均有“重置”按钮，一键清除筛选条件。

### 2.9 购买流程（从详情到支付）
- **问题**：从查看数码产品/配件详情到购买，完整流程是什么？
- **答案**：
  1. 查看详情：
     - 数码产品：列表页点击卡片跳转详情页；
     - 数码配件：列表页点击“查看详情”弹窗展示；
     （可查看名称、价格、库存、销售店铺等，数码产品还可查看品质，店铺可跳转）；
  2. 收藏操作：数码产品详情页点击“加入收藏”（星标图标），已收藏可点击“取消收藏”；
  3. 购买数码产品：
     - 选择收货地址（默认地址自动填充）；
     - 点击“立即购买”生成订单；
     - 支付抽屉点击“立即支付”（余额不足可跳转充值，支持100/200/500/1000元或自定义金额）；
     - 确认后完成购买；
  4. 加入购物车：配件列表页/详情弹窗点击“加入购物车”（图标），提示成功后可在购物车统一结算。


## 三、交易支付相关
### 3.1 购物车结算流程
- **问题**：如何在购物车中结算商品并完成支付？
- **答案**：购物车勾选目标商品→选择收货地址→点击“去结算”生成订单→支付抽屉点击“立即支付”确认；余额不足可通过充值窗口选择100/200/500/1000元或自定义金额（保留两位小数）充值后支付。

### 3.2 订单状态及操作
- **问题**：订单包含哪些状态？不同状态下可执行什么操作？
- **答案**：
  - 状态：待付款、待接单、派送中、已送达、已完成、已取消；
  - 操作：
    - 待付款/待接单：可取消订单；
    - 已送达：可确认完成订单；
    - 已完成/已取消（数码产品订单）：支持“再次购买”。

### 3.3 订单搜索
- **问题**：如何搜索特定的订单？
- **答案**：在“数码产品订单”或“数码配件订单”标签页搜索框输入订单号→点击“搜索”按钮，系统弹窗展示匹配订单详情（含商品信息、价格、状态等）。

### 3.4 支付时间限制
- **问题**：订单支付有时间限制吗？超时未支付该怎么处理？
- **答案**：支付有10分钟倒计时限制；超时未支付可在订单列表重新发起支付，或点击“放弃支付”关闭窗口（订单可能自动转为“已取消”状态）。

### 3.5 账户充值
- **问题**：账户如何充值？支持哪些金额？
- **答案**：支持100/200/500/1000元固定金额及自定义金额（保留两位小数）；通过个人资料页“充值”按钮或支付时的弹窗输入金额，点击“充值”完成操作。

### 3.6 取消订单退款
- **问题**：取消订单后款项如何处理？
- **答案**：未支付订单取消无资金变动；已支付订单取消后，款项自动退回用户账户余额（默认即时到账）。

### 3.7 收货地址管理
- **问题**：如何管理收货地址（新增/编辑/删除/设为默认）？
- **答案**：
  - 新增：点击“新增收货地址”，填写收货人、手机号、省市区、详细地址（支持预设标签“家/公司/学校”或自定义）；
  - 编辑/删除：在地址卡片点击对应按钮（删除需确认）；
  - 设为默认：勾选“设置为默认地址”复选框。

### 3.8 账户余额查看与到账时间
- **问题**：如何查看账户余额？充值后多久能到账？
- **答案**：
  - 查看余额：个人资料页“我的个人资料”板块（格式“￥XXX.XX”）；
  - 到账时间：充值成功后实时到账，余额即时更新。

### 3.9 购物车商品调整
- **问题**：购物车商品如何调整数量或移除？
- **答案**：
  - 调整数量：通过数字输入框增减（最小1），系统实时更新合计金额；
  - 移除商品：点击“移除购物车”按钮，确认后删除，列表自动刷新。


## 四、后台管理相关
### 4.1 新增在售数码产品
- **问题**：店铺在后台如何新增在售数码产品？需要填写哪些信息？
- **答案**：进入“数码产品管理”页面点击“新增”，填写以下必填信息：
  1. 基础信息：
     - 数码产品类型（下拉选择）；
     - 名称（2-20字）；
     - 性别（公/母）；
     - 价格（非负数字）；
     - 库存数量（非负整数）；
  2. 多媒体信息：数码产品图片（jpeg/png/webp/gif格式，≤2MB）；
  3. 描述信息：
     - 数码产品介绍（≤250字）；
     - 详细内容（富文本编辑器填写，支持图片插入）；
  4. 售卖状态：选择“已售罄”“售卖中”或“未上架”；
  提交保存后，数码产品自动进入店铺商品列表。

### 4.2 修改数码配件库存和售价
- **问题**：店铺如何修改数码配件的库存和售价？
- **答案**：
  1. 进入“商品管理”页面，通过“商品名称”或“类型”筛选目标数码配件；
  2. 找到对应商品，点击“编辑”按钮；
  3. 编辑弹窗中修改：
     - 剩余数量（非负整数）；
     - 商品价格（非负金额，保留两位小数）；
  4. 点击“确定”保存，用户端实时同步，库存为0时显示“缺货”标签。

### 4.3 订单信息查看与筛选
- **问题**：店铺在后台能看到哪些订单信息？如何筛选特定订单？
- **答案**：
  - 可查看信息：
    1. 数码产品订单：订单号、状态（待付款/待接单等）、下单时间、数码产品名称/图片/价格、收货人信息、用户昵称；
    2. 数码配件订单：在数码产品订单基础上增加“商品单价”“购买数量”“商品总价”；
  - 筛选方式：
    - “订单号”精准查询；
    - “订单状态”（如待接单）、“下单时间”（日期范围）筛选；
    筛选后支持分页查看。

### 4.4 处理“待接单”数码产品订单
- **问题**：用户下单后，店铺如何处理“待接单”状态的数码产品订单？
- **答案**：
  1. 在“数码产品订单”列表找到“待接单”状态订单（橙色标签）；
  2. 点击操作栏“接单”按钮，弹窗确认“是否接单”；
  3. 确认后订单状态更新为“派送中”，用户端收到状态变更通知；
  （注：仅“待接单”状态可执行接单操作）。

### 4.5 处理数码配件订单“售后退款”
- **问题**：店铺如何处理数码配件订单的“售后退款”申请？
- **答案**：仅“已送达”或“已完成”状态的数码配件订单支持售后退款：
  1. 在“商品订单”列表找到目标订单，点击“售后退款”按钮；
  2. 弹窗确认“是否进行售后退款操作”；
  3. 确认后系统执行退款，订单状态同步更新，款项退回用户账户，提示“退款成功”并刷新列表。

### 4.6 设置数码产品“推荐”状态
- **问题**：店铺如何将数码产品设置为“推荐”状态，让其在用户端优先展示？
- **答案**：
  1. 进入“数码产品管理”页面，找到目标数码产品；
  2. 若当前为“未推荐”，点击操作栏“设置推荐”按钮（带星星图标）；
  3. 系统调用接口设为推荐，成功后标签变为“推荐中”；
  （注：推荐数量建议≤12个，便于用户端排版）。

### 4.7 批量删除数码配件
- **问题**：店铺在后台如何批量删除不需要的数码配件？
- **答案**：
  1. 在“商品管理”表格勾选需删除的多个数码配件；
  2. 点击页面上方“批量删除”按钮，弹窗提示“删除操作不可逆”；
  3. 确认后系统删除选中数码配件，页面自动刷新。

### 4.8 设置数码产品“已售罄”状态（库存不足时）
- **问题**：当数码产品库存不足时，店铺如何快速将其设置为“已售罄”状态？
- **答案**：
  1. 进入“数码产品管理”页面，找到库存不足的数码产品；
  2. 点击“编辑”按钮，在“售卖状态”中选择“已售罄”；
  3. 点击“确定”保存，用户端显示“已售罄”标签，无法下单。

### 4.9 标记“派送中”订单为“已送达”
- **问题**：店铺如何在后台查看“派送中”状态的订单，并标记为“已送达”？
- **答案**：
  1. 在“数码产品订单”或“商品订单”列表，通过“订单状态”筛选“派送中”订单（蓝色标签）；
  2. 找到已完成派送的订单，点击操作栏“送达”按钮；
  3. 弹窗确认“是否送达”，确认后订单状态更新为“已送达”。
